DBMS Virtual Laboratory

The lab contents are chosen such that they cover the syllabi of Database Management System courses including:

  1. Introduction to DBMS - DBMS vs File System, Three Schema Architecture, Data Independence.
  2. Entity Relationship Model - ER Diagram Design, ER to Relational Mapping.
  3. Relational Model and SQL - Relational Algebra, SQL Queries, Joins and Subqueries.
  4. Database Design and Normalization - Functional Dependencies, Normal Forms (1NF to BCNF).
  5. Transaction Processing and Concurrency - ACID Properties, Serializability, Locking Protocols.
