Experiment 2

Three Schema Architecture

Aim

To study the Three Schema Architecture of DBMS.

Objective

  • Understand External Schema.
  • Understand Conceptual Schema.
  • Understand Internal Schema.
  • Learn Data Independence.

Theory

The Three Schema Architecture separates user applications from physical database storage. It consists of:

  • External Level
  • Conceptual Level
  • Internal Level
This architecture provides logical and physical data independence.
