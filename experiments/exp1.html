DBMS vs File System

Aim

To study and compare Database Management System and File System.

Objective

  • Understand DBMS.
  • Understand File System.
  • Compare advantages and disadvantages.
  • Analyze redundancy and consistency.

Theory

A File System stores data in individual files. A Database Management System stores data in structured tables and provides security, consistency, integrity and controlled access. DBMS minimizes redundancy and supports concurrent access.

Comparison Table

Feature File System DBMS
Redundancy High Low
Security Low High
Backup Manual Automatic
Concurrency No Yes
Integrity Low High

Procedure

  1. Study File System.
  2. Study DBMS.
  3. Observe features.
  4. Compare both systems.
  5. Analyze results.
