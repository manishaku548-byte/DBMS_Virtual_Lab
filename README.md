# DBMS Virtual Laboratory

Interactive virtual laboratory for Database Management System (DBMS) experiments, aligned with the AKTU curriculum.

## Features

- 10 interactive experiments covering DBMS fundamentals
- Theory, simulations, SQL labs, ER modeling, normalization, and more
- Responsive web-based learning platform

## Experiments

1. DBMS vs File System
2. Three Schema Architecture
3. ER Diagram Design
4. ER to Relational Mapping
5. Relational Algebra
6. SQL Query Laboratory
7. SQL Join Visualizer
8. Functional Dependency
9. Normalization
10. Transactions & Concurrency

## How to Run

Open `index.html` in a web browser. No server setup required.
