Experiment 1: DBMS vs File System

What is a File System?

A File System stores data in individual files. Each application manages its own files independently, which often leads to duplicated and inconsistent data across the organization.

What is a DBMS?

A Database Management System stores data in structured tables and provides security, consistency, integrity and controlled access. DBMS minimizes redundancy and supports concurrent access by multiple users.
