Virtual Database Management System Laboratory

The lab contents are chosen such that they cover the syllabus of Database Management System courses, including:

  1. Database Architecture and the Three Schema Architecture
  2. Entity Relationship Modeling and ER-to-Relational Mapping
  3. Relational Algebra and Relational Model Concepts
  4. SQL Programming, Queries and Join Operations
  5. Functional Dependencies and Candidate Key Analysis
  6. Normalization (1NF, 2NF, 3NF, BCNF)
  7. Transaction Processing and Concurrency Control
  8. Standard university DBMS laboratory curricula, including AKTU
